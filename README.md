# 🌟 Stern Pizza & Kebap Haus — Website

Eine moderne, responsive Restaurant-Website für das **Stern Pizza & Kebap Haus** in Volkertshausen, entwickelt als Praxisprojekt im Bereich Web-Entwicklung.

---

## 🔗 Live-Vorschau

> Projekt lokal öffnen: `index.html` im Browser aufrufen


---

## 🛠️ Tech Stack

| Technologie | Einsatz |
|---|---|
| **HTML5** | Semantische Seitenstruktur, SEO-Grundlagen |
| **CSS3** | Responsive Design, Animationen, Grid & Flexbox |
| **JavaScript (Vanilla)** | Tab-Navigation, Scroll-Animationen, Mobile Menu |

---

## ✨ Features

- **Responsive Design** — Mobile-first, funktioniert auf allen Bildschirmgrößen
- **Hero-Section** mit Grill-Textur-Effekt via CSS Pseudo-Elemente
- **Speisekarte** mit Tab-Navigation (Döner / Pizza / Burger)
- **Scroll-Animationen** mit der `IntersectionObserver` API
- **Sticky Navbar** die beim Scrollen schrumpft
- **Google Maps** Integration
- Farbpalette direkt aus dem Firmenlogo abgeleitet (Weinrot `#8B1A4A` + Gelbgrün `#A8C63D`)
